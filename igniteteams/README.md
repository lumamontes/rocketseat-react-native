## Ignite Teams

App desenvolvido durante o segundo módulo do curso de React Native da Rocketseat.
É possível cadastrar grupos e separar jogadores entre times.

Foi abordado conceitos como:
- Styled Components
- React Navigation
- Async Storage
- Typescript
